# MySQL Connection Options to Try

Option 1: Using Docker network (try this first)
spring.datasource.url=jdbc:mysql://host.docker.internal:3306/unimart

Option 2: Using the container name (if your Docker container is named "db")
spring.datasource.url=jdbc:mysql://db:3306/unimart

Option 3: Using Docker's default gateway
spring.datasource.url=jdbc:mysql://172.17.0.1:3306/unimart

Option 4: Using your machine's IP address
spring.datasource.url=jdbc:mysql://192.168.1.X:3306/unimart
(Replace X with your local IP address - find with ipconfig/ifconfig)

Option 5: If your MySQL container is exposed on a different port
spring.datasource.url=jdbc:mysql://localhost:33306/unimart
(If you mapped port 3306 to 33306 in docker-compose.yml)
